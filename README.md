# Esports express

这是一份大英的期末小组作业, 网站基于bootstrap框架上的jekyll, 发布在GitPage上. 

请各位同学根据markdown语法自行修改`./_posts`中自己文章的内容. `.md`, `.markdown`文件可以使用任何文本编辑器打开, 但是推荐使用有内置markdown支持的VScode或者其他的markdown专门编辑工具. 语法...略. 

Todolist:

* 提交完整文章
* 配图!!!!!!
* 交换文章进行语法和逻辑检查
* 尽量早点提交初稿
* 根据反馈意见进行修改
* 提交终稿
